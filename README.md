# vue-Meizi

>  本项目是基于vue2最新实战项目，是适合新手练习.实现了移动端使用最多的 无限滚动，图片加载，左右滑动。


### 项目技术架构
***
*  vue-cli
*  vue
*  vue-resource
*  vue-router
*  vuex
*  vue-awesome-swiper
*  vue-infinite-scroll
*  stylus
*  webpack

###安装
***
项目地址：（`git clone`）
```shell
git clone https://github.com/chongwenwen/vue-text2.git
```
通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```
启动服务(http://localhost:9090)

```
npm run dev
```
发布代码

```
npm run build
```
